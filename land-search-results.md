## 📋 Manual Search Checklist

When reviewing listings, capture the following for each property:

- [ ] **Price:** List price
- [ ] **Acreage:** Exact acreage
- [ ] **Location:** Full address or nearest town/city
- [ ] **County:** Which county
- [ ] **URL:** Link to listing
- [ ] **Road Access:** ✓ if mentioned, ? if unclear, ✗ if none
- [ ] **Zoning:** Flag if ag-only, recreational-only, or conservation easement
- [ ] **Utilities:** Note if power, water, septic mentioned
- [ ] **Notes:** Any other relevant details

## 🔄 Alternative Approaches

If automated scraping is required:

1. **Residential Proxy Services:** Use rotating residential proxies (Bright Data, Smartproxy, etc.) to avoid bot detection
2. **CAPTCHA Solving Services:** 2Captcha or Anti-Captcha to bypass verification
3. **Selenium with Undetected ChromeDriver:** Python library that bypasses bot detection
4. **API Access:** Contact sites directly for API access (usually paid)
5. **MLS Integration:** Work with a real estate agent who has MLS access
6. **Zillow/Realtor.com:** Try alternative sites that may have weaker protections
7. **County Assessor Records:** Public records available through county websites
